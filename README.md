# backend-Authentication-system

🔐 Authentication System

A production-ready authentication system built with Node.js, Express.js, MongoDB, and JWT.
This project demonstrates secure authentication architecture used in real-world applications, including token-based authentication and session management.

🚀 Features

User Registration & Login

JWT Access Token Authentication

Refresh Token System

Token Rotation

Session Management

Logout from Current Device

Logout from All Devices

Protected Routes with Middleware

🛠️ Tech Stack

Node.js

Express.js

MongoDB & Mongoose

JWT (JSON Web Token)

📡 API Endpoints

POST /register – Create account

POST /login – User login

GET /refresh – Refresh access token

GET /logout – Logout current session

GET /logout-all – Logout from all devices

GET /GetMe – Protected route
